# Raven Nuker
Raven Nuker is a discord server nuker.

# Features
- Create Channels
- Delete Channels
- Ban Member
- Create Random Roles
- Delete All Roles
- Set Server Name
- Spam @everyone On All Channels

# Usage
- `$ pip install requests colorama threading random string`
- `$ git clone https://github.com/fvtihakar/RavenNuker`
- `$ cd RavenNuker`
- `$ python3 main.py`
